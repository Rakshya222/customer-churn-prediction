# Customer Churn Prediction (Telecom)

## Overview
This project predicts customer churn using telecom customer data with Logistic Regression, Random Forest, and XGBoost.
The goal is to identify customers likely to leave and understand key factors influencing churn.

## Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding
- Model building and evaluation
- Feature importance analysis

## Key Findings
- Month-to-month contracts and electronic check payment methods have higher churn
- XGBoost achieved 74% accuracy with 77% recall for churned customers
- Top features identified actionable insights for business retention

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, XGBoost

## Dataset
Source: [Kaggle Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
